<h1>YOLOPHP</h1>

<h3>YOLOv8 functionality on a PHP server using the PHP FFI and LibTorch.</h3>

<h2>Directory Structure<h2>

_________________

```
├── CMakeLists.txt              # Build options and configuration for C++ library
├── README.md                               
├── build                       # Directory for build outputs generated by CMake
│   ├── CMakeCache.txt
│   ├── CMakeFiles
│   │   ├── ...
│   ├── ... 
│   └── libYOLO.so              # # Compiled shared library for YOLO (output of the build process)
├── include                     # Directory containing header files for the project
│   ├── stb                     # Third-party single-file public domain libraries for image processing
│   │   ├── stb_image.h
│   │   ├── stb_image_resize.h
│   │   └── stb_image_write.h
│   └── yolov8.h
├── libtorch                    # Directory containing the libtorch library files
│   ├── ...
├── model                       # Directory for model-related files 
│   ├── modelExport.py          # Python script for exporting YOLOv8 model into TorchScript
│   ├── yolov8n.pt              # YOLOv8 model downloaded from ultralytics website
│   └── yolov8n.torchscript     # TorchScript equivalent of "yolov8n.pt"
├── notebooks                   # Jupyter notebooks for experimentation
│   └── outputs.ipynb           # Notebook to model the post-processing steps needed
├── output                      # Directory for generated outputs
│   └── output.jpg              # Example output image generated by the system
├── public                      # Directory for files to be served publicly (web server files)
│   └── index.php               # Example PHP file for web interface
├── src                         # Source files for the C++ library
│   ├── stb_image_impl.cpp      # Implementation file for stb image library functions
│   └── yolov8.cpp              # Main source file for YOLOv8 functions
└── testInput.jpg               # Example input image for testing
```
